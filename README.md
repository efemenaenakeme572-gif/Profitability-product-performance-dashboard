# profitability-product-performance-dashboard
Interactive Power BI dashboard providing insights into sales performance, profit trends, and product-level analysis

📊 Profitability & Product Performance Dashboard
🚀 Project Overview
This project showcases my ability to analyze business data and deliver actionable insights using Power BI.
The dashboard provides a comprehensive view of sales performance, profitability, and product-level trends, enabling stakeholders to make informed, data-driven decisions.

🎯 Objective
To design an interactive dashboard that:
Identifies high-performing and low-performing products
Tracks profit trends over time
Highlights key business drivers of profitability
Supports strategic decision-making

💼 Business Context
Organizations often generate high sales but struggle with profitability visibility. This project simulates a real-world scenario where decision-makers need to:
Understand which products drive profit
Monitor financial performance across categories
Identify opportunities for cost optimization and growth

📈 Key Insights
Total Sales: 324.28K
Total Profit: 34.57K
Profit Margin: 11%
🔍 Insights Derived:
Office Supplies contribute the highest overall profit, despite not always leading in sales
Profit trends show steady growth, with peaks towards the end of the year
A small group of products accounts for a significant share of total profit
Some categories generate strong sales but lower profitability, highlighting optimization opportunities

🧠 Analytical Approach
Cleaned and transformed raw data using Power Query
Created calculated measures (e.g., Profit, Profit Margin) using DAX
Designed an intuitive and interactive dashboard for easy exploration of insights
Focused on clear storytelling and business relevance
