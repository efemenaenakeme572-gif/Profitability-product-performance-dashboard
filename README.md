# Profitability-product-performance-dashboard
Power BI dashboard analyzing sales,profitability,and product performance
🧾 Project Overview
This project showcases an interactive Power BI dashboard built to analyze sales performance, profitability, and product-level insights.
The dashboard helps stakeholders quickly understand how different product categories and individual products contribute to overall business performance.
🎯 Business Problem
Businesses often struggle to:
Identify which products are truly profitable
Understand trends in profit over time
Distinguish between high sales and high profitability
This dashboard addresses these challenges by providing a clear, visual breakdown of financial performance and product impact.

📈 Key Insights

A small number of products drive a large portion of profit
Profitability fluctuates monthly, indicating potential seasonality or cost variation
Technology category is the most profitable segment
